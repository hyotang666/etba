## 仕様
### 実装済み
* プレイヤーは一人。
* 敵も一人。
* <del>攻撃種類も一つ。</del>
* 画面は固定。
* プレイヤーのHPがゼロになったらゲームオーバー。
* 敵のHPがゼロになったらクリア。

### 追加仕様
* カーソルキーを連続入力することで移動速度アップ。
* 攻撃ボタンを押しっぱなしにすることでチャージ攻撃。
  * チャージの溜まり具合で攻撃パターン切り替え。
* 攻撃ボタンを連打することでいわゆる百烈張り手的な攻撃。
* 攻撃ボタンを一定のテンポで入力することで前進を伴うコンボ追撃。
